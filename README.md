# csv-coding-task

## Installation

  1. cd into the csv-coding-task directory in the command line
  1. run `pip install -r requirements.txt`

## Running the code

  1. cd into the csv-coding-task directory in the command line
  1. run `python parse.py`

## Running tests

  1. cd into the csv-coding-task directory in the command line
  1. run `pytest`
